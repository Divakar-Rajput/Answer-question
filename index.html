10. ____ addressing mode is most suitable to change the normal sequence of execution of instructions.
a) Relative
b) Indirect
c) Index with Offset
d) Immediate

Answer: a
Explanation: The relative addressing mode is used for this since it directly updates the PC.

11. Which of the following is not a valid addressing mode?
a) Register
b) Absolute
c) Immediate
d) Floating-Point

Answer: d
Explanation: Floating-Point is a data type, not an addressing mode.
